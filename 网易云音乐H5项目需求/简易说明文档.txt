
功能：
	注册
		-> 注册用户
	登陆
		-> 用户登陆
	
	主页
	     :音乐标签页
		-> 搜索
		-> 推荐歌单
		-> 最新音乐
		-> 推荐MV
	     :视频标签页

	歌单
		-> 根据当前点击歌单，显示歌单详细信息及歌曲列表
		

	播放音乐
		-> 显示当前选中音乐，功能，播放/暂停/上一首/下一首，播放时音乐封面会转动，停止则停
		-> 音乐进度条，实现功能（根据当前播放的秒数百分比计算然后滚动小球）
		-> 每次播放的歌曲，会被记录到数据库内（最近播放 此功能需要使用这些数据）
		（播放音乐时，其实就是播放歌单内的音乐，上一首，下一首，其实就是在歌单内切换上一首下一首）
		（随机播放音乐，无论点击上一首下一首，都直接从歌单内随机一首）


	我的音乐
		-> 最近播放（功能见图片描述）
		-> 最近播放列表  （点击任意一首，立刻切换到播放音乐界面，并播放此音乐）

	账号
		-> 当用户未登陆时，提醒登陆，登陆以后，显示用户头像和名字
		-> 点击用户头像，可以进入到更换头像界面，选取一张图片后，可更换当前头像
		-> 登陆和注销按钮在此界面最下方
